# Contact Form & To-Do List

## 📌 Project Overview
This project was developed as **Task 2** of the **Web Development Internship** at **ApexPlanet Software Pvt. Ltd.**

The project demonstrates HTML, CSS, and JavaScript concepts by creating a responsive Contact Form with validation and a dynamic To-Do List using DOM manipulation.

---

## 🚀 Features

### 📧 Contact Form
- Full Name, Email, Phone Number, and Message fields
- JavaScript form validation
- Email format validation
- 10-digit phone number validation
- Success alert after valid submission

### ✅ To-Do List
- Add new tasks
- Delete tasks
- Dynamic DOM manipulation

### 🎨 Responsive Design
- CSS Grid layout
- Responsive design using Media Queries
- Clean and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
